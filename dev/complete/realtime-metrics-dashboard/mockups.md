# Dashboard UI Mockup - Plan-Centric Sidebar

**Philosophy**: Plan orchestration is front and center. The dashboard is designed around monitoring multiple concurrent plan executions, with metrics and session history as supporting context.

---

## Navigation Structure

```
Sidebar:
  - Overview (metrics + active plans summary)
  - Plans (detailed plan management & history)
  - Sessions (historical session review)
  - Settings
```
